# Airbnb Clone – Full Stack Web Application (MERN Stack)

A fully functional **Airbnb Clone** built using the **MERN Stack (MongoDB, Express.js, React.js, and Node.js)**. This project allows users to browse property listings, create and manage accommodations, and securely authenticate using modern web technologies.

Features

* User Authentication & Authorization (JWT)
* Browse and Search Property Listings
* Add, Edit, and Delete Listings
* Image Upload Support
* Responsive UI for Mobile & Desktop
* Secure RESTful APIs
* MongoDB Database Integration
* Error Handling & Form Validation

 Tech Stack

* Frontend:React.js, HTML5, CSS3, JavaScript, Bootstrap
* Backend:Node.js, Express.js
* Database:MongoDB
* Authentication:JWT
* Version Control:Git & GitHub

 Learning Outcomes

* Built a complete full-stack web application using the MERN Stack.
* Implemented secure authentication and CRUD operations.
* Improved API development, database management, and responsive UI design.
* Gained hands-on experience with full-stack development and deployment.
